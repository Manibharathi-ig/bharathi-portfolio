#  Bharathi Portfolio

A modern full-stack portfolio application built to showcase my professional experience, technical skills, and projects. The application includes a public portfolio website, contact management system, secure admin dashboard, and cloud-based deployment infrastructure.

##  Live Demo

Frontend: https://bharathi-portfolio-two.vercel.app

##  Features

### Public Portfolio

* Modern responsive design
* Professional landing page
* Skills showcase
* Project showcase
* Experience section
* Contact form
* GitHub and LinkedIn integration

### Admin Dashboard

* Secure admin login
* Contact submissions management
* View all contact requests
* Update contact status
* Delete contact requests

### Backend Features

* REST API architecture
* Form validation
* PostgreSQL database integration
* Prisma ORM
* Secure password hashing with bcrypt
* Admin authentication

##  Tech Stack

### Frontend

* React
* TypeScript

* React Router
* Axios

### Backend

* Node.js
* Express.js
* TypeScript
* Prisma ORM
* PostgreSQL
* bcrypt

### Database

* Neon PostgreSQL

### Deployment

* Vercel (Frontend)

##  Authentication

The application includes a secure admin authentication system using:

* bcrypt password hashing
* Protected dashboard access
* PostgreSQL user storage

##  Database

The project uses Neon PostgreSQL with Prisma ORM for:

* Contact management
* Admin authentication
* Data persistence

##  Deployment Architecture

Frontend → Vercel

Backend API → Render

Database → Neon PostgreSQL
